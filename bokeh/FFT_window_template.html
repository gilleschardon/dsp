Analyse spectrale par FFT - fenêtrage

En présence de plusieurs sinusoïdes d'amplitudes variées, les pics des sinusoïdes de faible puissance peuvent être cachés par les lobes secondaires des sinusoïdes de plus grande puissance, même si la durée d'observation est suffisante pour les résoudre.

L'utilisation de fenêtres permet de diminuer la hauteur des lobes secondaires des sinusoïdes de fortes puissances et de révéler les lobes principaux des sinusoïdes moins puissantes. De façon générale, ce fenêtrage élargit les lobes principaux, et diminue les performances de l'analyse en terme de résolution.

La démonstration propose de comparer la fenêtre rectangulaire \[R[n] = 1 \mbox{ pour } 0 \leq n < N \] la fenêtre de Hann \[H[n] = \frac{1}{2}\left(1 - \cos\left(\frac{2\pi n}{N-1}\right)\right) \mbox{ pour } 0 \leq n < N, \] et la fenêtre de Hamming \[H[n] = \left(0.54 - 0.46 \cos\left(\frac{2\pi n}{N-1}\right)\right) \mbox{ pour } 0 \leq n < N. \]
